# Support

## Stuck on Something?

Don't worry — everyone gets stuck. I am here to help.

If you are working on a contribution and something is not making sense —
whether it is the codebase, the agent config, the system prompt, or anything else —
just drop a comment on the issue you are working on.

I check GitHub every day and I will reply within **24 hours**. No question is too small.

---

## How to Ask for Help

Just comment on the issue you are working on with something like:

> "Hey, I am stuck on [this part]. Here is what I tried — [what you did]. Can you help?"

That is it. No need to be formal. Just tell me where you are stuck and I will help you move forward.

---

## What I Can Help With

- Understanding the codebase
- Writing a good system prompt for your agent
- Figuring out which input or output type to use
- Debugging why your agent is not working
- Any general question about the project

---

## Please Don't

- Open a new issue just to ask a question — comment on the existing one instead
- Send DMs for support — keeping it public helps others who have the same question
